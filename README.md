# Tour App #

Concept for an app used by touring professionals to have access to all information for a venue and the city or area the venue is located at.

## Technologies ##
[x] JavaScript
[x] Express
[x] MongoDB
[x] Mongoose.js
[x] Angular
[x] Twitter Bootstrap
[x] HTML5
[x] CSS
[x] AWS - EC2
